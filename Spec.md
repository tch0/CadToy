# 项目描述

## 概述

- 项目目的：实现一个类似于AutoCAD的极简版本，主要功能包括2D绘图、基本的几何操作（如绘制直线、圆、矩形等）、变换（如平移、旋转、缩放）、图层管理、颜色设置等。
- 项目技术栈：
  - 前端：OpenGL + Dear ImGui
  - 后端：C++
- 项目架构：
  - 模块划分：
    - 渲染模块：负责绘制2D图形。
    - 输入模块：处理鼠标、键盘输入。
    - 几何模块：实现基本的几何操作（如计算点、线、圆的交点等）。
    - 变换模块：处理图形的平移、旋转、缩放。
    - 图层模块：管理不同图层上的图形。
    - 颜色模块：设置和管理图形的颜色。
    - 撤销/重做模块：实现撤销和重做操作。
    - 保存/加载模块：实现图形的保存和加载。
    - 主循环模块：负责主循环的管理。
    - 事件处理模块：处理用户输入事件。
    - 工具模块：提供一些辅助工具（如测量工具、对齐工具等）。
    - 帮助模块：提供用户帮助和文档。
    - 日志模块：记录运行时的日志信息。
    - 配置模块：管理项目的配置参数。
    - 资源模块：管理项目的资源（如字体、图标等）。
    - 内存管理模块：负责内存的分配和释放。
    - 性能优化模块：优化渲染性能和响应时间。
- 项目功能：
  - 2D绘图：支持绘制直线、圆、矩形等基本几何图形。
  - 变换操作：支持平移、旋转、缩放等变换操作。
  - 图层管理：支持创建、删除、切换图层。
  - 颜色设置：支持设置和管理图形的颜色。
  - 撤销/重做：支持撤销和重做最近的操作。
  - 保存/加载：支持保存当前图形到文件，以及从文件加载图形。
- 相关功能命令：
  - 命令行界面：用户可以通过输入英文命令来执行操作（如绘制直线、圆等）。
  - 命令解析模块：负责解析用户输入的命令，并调用相应的模块来执行操作。
  - 命令列表：
    - 绘制直线：`LINE X1 Y1 X2 Y2`
    - 绘制圆：`CIRCLE X Y RADIUS`
    - 绘制矩形：`RECT X Y WIDTH HEIGHT`
    - 平移：`TRANSLATE X Y`
    - 旋转：`ROTATE X Y ANGLE`
    - 缩放：`SCALE X Y SCALE_FACTOR`
    - 创建图层：`LAYER NAME`
    - 删除图层：`DELETE_LAYER NAME`
    - 切换图层：`SWITCH_LAYER NAME`
    - 设置颜色：`COLOR R G B`
    - 撤销：`UNDO`
    - 重做：`REDO`
    - 保存：`SAVE FILE_PATH`
    - 加载：`LOAD FILE_PATH`
    - 退出：`EXIT`
    - 帮助：`HELP`
    - 日志：`LOG`
    - 配置：`CONFIG`

## 文件创建相关注意事项

- 所有文件都是用UTF不带BOM格式，使用CRLF作为换行符。
